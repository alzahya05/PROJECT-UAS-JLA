# Redesign: Modern Dark Premium Landing Page

## Overview
Merestyling seluruh UI menjadi landing page modern premium dengan tema dark, gradasi biru-pink, efek glow, glassmorphism, dan visual futuristik. **Tanpa mengubah** struktur, routing, logic, state, function, konten, atau data.

## Approach
Menggunakan pendekatan **CSS-first** (Tailwind v4): mengubah color tokens di `globals.css` + menambah utility classes di setiap komponen. Semua perubahan hanya pada styling (className dan CSS variables).

---

## File Changes

### 1. `app/globals.css` ÔÇö Color Tokens & Utilities
**Perubahan inti:**
- Force dark mode: pindahkan dark token ke `:root` langsung (remove `@media (prefers-color-scheme: dark)`)
- Update color palette: dark backgrounds (#050510, #0a0a1a, #111127), primary = blue gradient (#6366f1 Ôćĺ #3b82f6), accent = pink (#ec4899, #f472b6)
- Tambah CSS custom properties untuk glow effects: `--glow-blue`, `--glow-pink`
- Tambah utility classes via `@layer utilities`:
  - `.glow-blue` ÔÇö box-shadow blue glow
  - `.glow-pink` ÔÇö box-shadow pink glow
  - `.glow-dual` ÔÇö combined blue+pink glow
  - `.glass` ÔÇö glassmorphism (backdrop-blur + semi-transparent bg)
  - `.gradient-text` ÔÇö blue-to-pink gradient text
  - `.gradient-border` ÔÇö gradient border effect (via border-image or pseudo-element)
  - `.card-premium` ÔÇö shared card styling (dark bg, thin border, shadow)
  - `.btn-primary-gradient` ÔÇö gradient button (blueÔćĺpink)
  - `.btn-secondary-glass` ÔÇö glass button
  - `.section-glow` ÔÇö subtle radial gradient background for sections

### 2. `app/layout.tsx` ÔÇö Root Layout
**Perubahan:**
- Tambah `dark` class ke `<html>` untuk force dark mode
- Body: tambah `bg-background text-foreground` explicit classes

### 3. `app/page.tsx` ÔÇö Homepage (Hero, Modules, Quick Access, CTA)
**Hero Section:**

**Module Cards:**

- Border: `border-white/[0.08]` Ôćĺ subtle white border
- Background: `bg-[#0c0c1d]` or use card token
- Icon containers: gradient backgrounds (blue/pink/purple gradient combos)
- Feature arrows: pink accent color

**Quick Access Cards:**
